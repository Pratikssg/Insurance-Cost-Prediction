About This Project

This project is a demonstration of an end-to-end Machine Learning Operations (MLOps) workflow, designed to be reproducible, scalable, and maintainable.

Key Technologies Used:

  • Machine Learning: Python with Scikit-Learn for model training (Random Forest Regressor) and a robust preprocessing pipeline.
  • Backend API: A high-performance API built with FastAPI, using Pydantic for data validation.
  • Containerization: Docker for packaging the entire application and its dependencies into a portable image.
  • Cloud Deployment: Amazon Web Services (AWS), using ECR to store the Docker image and EC2 to host the live application.
